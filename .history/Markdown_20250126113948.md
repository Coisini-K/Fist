# 常用 npm 命令

## 初始化项目

### 初始化一个新的 package.json 文件，交互式提示用户输入信息

    npm init

### 使用默认设置快速初始化一个 package.json 文件

    npm init -y

## 安装依赖

### 安装指定的 npm 包，并将其添加到 dependencies 中

    npm install express

### 安装指定的 npm 包，并将其添加到 devDependencies 中（开发环境依赖）


```

    npm install nodemon --save-dev


```
